# GantangFinder — Daily Workflow Guide

*Step-by-step guide untuk execute marketing setiap hari*

---

## 🌅 DAILY WORKFLOW

### 🕐 09:00 — PAGI (Morning Check)

```
□ Buka TikTok → Cek notifikasi & analytics kemarin
□ Buka Facebook → Cek notifications & messages
□ Buka WhatsApp → Cek grup messages
□ Buka Instagram → Cek DM & comments

□ Review today's schedule dari calendar
□ Prepare content untuk hari ini
□ Check kalau ada event yang butuh countdown
```

---

### 🕐 12:00 — SIANG (Posting Session 1)

```
□ POST TIKTOK:
  - Upload video sesuai schedule
  - Copy caption dari content drafts
  - Add hashtags (Set 1-10 sesuai content type)
  - Post & share link

□ POST FACEBOOK (Round 1):
  - Post di 5 target groups
  - Use template dari content drafts
  - Include link app
  - Minimum 5 groups
```

---

### 🕐 14:00 — SIANG (Engagement)

```
□ TIKTOK:
  - Reply semua comments di videos
  - Like new comments
  - Share to WhatsApp if needed

□ FACEBOOK:
  - Reply comments di posts
  - Engage dengan other posts (like + comment)
  - Jangan lupa like posts orang lain

□ INSTAGRAM:
  - Like & comment di posts target accounts
  - Follow new accounts
  - Engage dengan bird community
```

---

### 🕐 17:00 — SORE (Posting Session 2)

```
□ POST FACEBOOK (Round 2):
  - Post di 5 target groups lagi
  - Total: 10 groups hari ini
  - Different content type dari Round 1

□ WHATSAPP:
  - Forward jadwal event ke semua grup WA
  - Share hasil event kalau ada
  - Format: Short message + link app

□ INSTAGRAM:
  - Post story kalau ada update
  - Story poll / question
```

---

### 🕐 20:00 — MALAM (Wrap Up)

```
□ ANALYTICS CHECK:
  - TikTok: View count, likes, followers
  - Facebook: Reach, engagement
  - WhatsApp: Response rate

□ SAVE CONTENT IDEAS:
  - Note down ideas untuk besok
  - Screen recording videos yang bagus
  - Save masteran recordings baru

□ PLAN TOMORROW:
  - Check calendar
  - Prepare content draft
  - Set reminders

□ DAILY LOG:
  - Catat metrics hari ini
  - Note what worked / didn't work
  - Plan improvement
```

## 📋 PLATFORM-SPECIFIC WORKFLOW

### TIKTOK WORKFLOW

```
1. CONTENT CREATION:
   □ Determine content type (masteran/tips/countdown)
   □ Open content drafts file
   □ Copy draft caption
   □ Select hashtags set
   
2. VIDEO:
   □ Record / select video footage
   □ Edit with CapCut / InShot
   □ Add text overlay
   □ Add music/sound
   
3. POSTING:
   □ Write caption (draft + link + hashtags)
   □ Post with optimal time (12:00 atau 17:00)
   □ Monitor views for 1-2 hours
   
4. ENGAGEMENT:
   □ Reply comments within 1 hour
   □ Like comments
   □ Duet/React if relevant
```

### FACEBOOK WORKFLOW

```
1. POSTING:
   □ Open Facebook Groups list
   □ Open content drafts file
   □ Select template sesuai content type
   □ Customize dengan event details
   
2. POSTING RULES:
   □ Max 2 posts per group per minggu
   □ Rotate groups (jangan spam same group)
   □ Always include link app
   □ Write dalam bahasa Indonesia
   
3. ENGAGEMENT:
   □ Reply all comments
   □ Answer DMs
   □ Engage di other posts (build relationship)
   
4. GROUP ROTATION:
   □ Week 1: Groups 1-10
   □ Week 2: Groups 11-20
   □ Week 3: Groups 1-10 (rotate back)
```

### WHATSAPP WORKFLOW

```
1. MESSAGE PREPARATION:
   □ Short message (max 500 chars)
   □ Include: Event name, date, link
   □ Format template:
      "🔔 Jadwal Lomba Minggu Ini
      
      📌 [Event Name]
      📅 [Date]
      📍 [Location]
      
      Daftar: [Link]"
   
2. SENDING:
   □ Forward ke semua grup WA
   □ Jangan spam (1x per grup per hari)
   □ Add personal touch
   
3. RESPONSE:
   □ Reply semua messages
   □ Answer questions about events
   □ Share additional info if needed
```

## 🔧 TROUBLESHOOTING

### Problem: TikTok video tidak dapat views
```
Solution:
1. Check hashtags - terlalu umum?
2. Cek timing - posting jam lain
3. Check sound - use trending sounds
4. A/B test different content types
```

### Problem: Facebook group blocked/removed
```
Solution:
1. Jangan spam - max 2x per group
2. Participate dulu sebelum promo
3. Bikin value post, bukan ads
4. Join backup groups
```

### Problem: WhatsApp di-flag spam
```
Solution:
1. Jangan forward terlalu sering
2. Add personal message
3. Vary content (schedules + results + tips)
4. Personal chat dulu, baru grup
```

### Problem: Tidak ada engagement
```
Solution:
1. Reply all comments (even negative)
2. Ask questions in posts
3. Use polling/survey format
4. Engage dengan audience posts first
```
