## 👥 FACEBOOK

### Option 1: Personal Account + Groups (Recommended)

1. Buat/gunakan Facebook personal account
2. Setup profile lengkap:
   ```
   Nama: GantangFinder
   Bio: Platform jadwal lomba burung kicau Indonesia
   ```

3. Gabung Facebook Groups (list ada di `02-facebook-groups.md`)

### Option 2: Facebook Page

1. Buka https://business.facebook.com
2. Buat Page baru
3. Setup:
   ```
   Page Name: GantangFinder
   Category: Website
   Bio: Platform jadwal lomba burung kicau Indonesia
   ```
