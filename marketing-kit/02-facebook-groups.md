## 📋 POSTING SCHEDULE

### Per Group Rules:
- Max 2x posting per minggu
- Jangan spam
- Selalu include link app

### Priority Posting:
1. **Jadwal Event** - 2x/minggu (Senin & Kamis)
2. **Hasil Event** - setelah event selesai
3. **Engagement** - 1x/minggu (Rabu/Senin)

### Template Posting:
```
🏆 EVENT LOMBA BURUNG 🏆

📌 [Nama Event]
📅 [Tanggal]
📍 [Lokasi]
🎯 Level: [Latber/Latpres/Regional/Nasional]
🐦 Kategori: [Jenis burung]
💰 Biaya: [Biaya pendaftaran]

Daftar sekarang di GantangFinder 👇
[LINK]

Jangan sampai kelewatan! 💪🔥
```
